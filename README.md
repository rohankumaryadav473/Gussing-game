# Number Guessing Game – Project Description

The Number Guessing Game is a simple Python project in which the computer randomly selects a number within a specified range, and the player tries to guess it. After each guess, the program provides hints such as "Too High" or "Too Low" until the correct number is guessed.

This project helps beginners learn Python programming concepts such as random number generation, loops, conditional statements, user input, and basic game logic.

## Features

* Randomly generates a secret number.
* Accepts user guesses from the keyboard.
* Provides hints ("Too High" or "Too Low") after each incorrect guess.
* Counts the number of attempts taken to guess the correct number.
* Displays a congratulatory message when the player wins.

##Technologies Used

* Python
* `random` module

# Applications

* Learning basic Python programming.
* Understanding loops and conditional statements.
* Practicing user input handling and problem-solving skills.
* Developing simple interactive console games.
